RECOUNT TEXT



Definition:


A text which retells events or experiences in the past. The purpose of recount text is to entertain or inform the readers.



Generic Structure:



Identification

It is a part that contains introduces or identifies the characterr that will be described.


Description

It is a part that contains an explanation, describe, characteristic related the topic.



Generic Structure:


1. Orientation : tells who was involved, what happened, where the events took place, and when it happened.


2. Events : Tells what happened and in what sequence.


3. Reorientation : Consists of optional-closure of events/ending.



Language Features:

1. Introducing personal participant: I, my friends, etc.
2. Using simple past tense.
3. Using action verb: wrote, bought, went, slept, etc.
