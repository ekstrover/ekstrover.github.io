DESCRIPTIVE TEXT



Definition:


A text which describing a person, thing, animal or place. The purpose is to describe a particular person, place, thing, etc.



Generic Structure:



Identification

It is a part that contains introduces or identifies the characterr that will be described.


Description

It is a part that contains an explanation, describe, characteristic related the topic.



Language Features:


- using simple present tense
- using adjective, noun (as a subject)
