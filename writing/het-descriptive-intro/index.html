DESCRIPTIVE TEXT



Purpose:


To describe or inform a particular thing to the reader. It can be person, plant, animal, etc.

Generic Structure:


- Identification : this part will describe us about general information of the topic.
- Description : this part will describe us about specific information of the topic.

Tenses:


When you write a descriptive text, you have to use simple present tense.
